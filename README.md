# Installation

```commandline
mkdir oiiai
cd oiiai
python3 -m venv .venv --prompt oiiai
. .venv/bin/activate

pip install git+https://github.com/dickerdackel/oiiai
# Will pull in pygame-ce, lfo, rpeasings, pgcooldown as dependencies

hash -r
oiiai
```
